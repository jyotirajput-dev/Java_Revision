🧠 Java Program: Datatypes and Variables

This program explains variables, data types, and naming conventions in Java.

📚 Concepts Covered
🔹 1. Variable

A variable is a memory location used to store data.

Example:

int age = 20;

int → Data type

age → Variable name

= → Assignment operator

20 → Value stored in memory

🔹 2. Data Type

Data type defines what kind of value a variable can store.

Example:

int → integer values

float → decimal values

char → single character

boolean → true/false

🔹 3. Rules for Variables

Cannot start with a number

Case-sensitive

No spaces allowed

Cannot use keywords

Allowed: letters, digits, _, $

🔹 4. Naming Conventions
✔ PascalCase (for Classes)

Used for class names:

MyClass

MyEmployee

MyLearningJourney

✔ camelCase (for Variables & Methods)

Used for variables and methods:

myScanner

myEmployee

myLearningJourney

🚀 Goal of This Program

To understand how data is stored using variables and how to write clean Java code using proper naming rules.
