☕ Java Program: Hello World

This is my first Java program that prints "Hello World" on the screen.
It helps in understanding the basic structure of a Java program.

Concepts Used

Package

Class

Main Method

Output using System.out.println()

Program Structure Explained

1. Package

package Practice1;

A package is used to organize related classes together.
It works like a folder that keeps Java files structured.

2. Class

public class Hello_World

A class is the blueprint of a Java program.
Every Java program must be written inside a class.

3. Main Method

public static void main(String[] args)

The main() method is the entry point of a Java program.
Execution of the program starts from here.

4. Printing Output

System.out.println("Hello World");

This statement prints text on the screen.

System → Java system class

out → Standard output stream

println() → Prints text and moves to the next line

Output
Hello World
